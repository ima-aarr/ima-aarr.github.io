# 負荷試験エンジン Pro v3.0

ブラウザ(Web Workers)の性能を限界まで引き出す、純国産の静的型負荷試験ツールです。
GitHub Pagesに配置するだけで、どこからでも高負荷テストが実行可能です。

## 実装のこだわり (High-Performance)
- **非同期バッチ処理**: 各Workerが `Promise.allSettled` を使用し、ネットワーク帯域を埋め尽くすようにリクエストをバースト送信します。
- **コネクションの最適化**: 応答待ちによるスレッド停止を避けるため、2秒のタイムアウト処理を実装。
- **マルチスレッド**: メインUIをフリーズさせず、マルチコアCPUのパワーを最大限に活用します。

## 使い方
1. `index.html` をGitHubリポジトリにアップロード。
2. Settings > Pages で公開設定をオンにする。
3. 生成されたURLにアクセスし、対象URLを入力して「攻撃開始」をクリック。

## 技術的制限
- **Mixed Content**: HTTPSサイトからHTTPサイトへのテストはブラウザによりブロックされます。
- **CORS**: `no-cors`モードを使用しているため、レスポンスの詳細（ステータスコード等）は取得できません。リクエストの送信成否のみカウントします。
